           Anti-Training Music Protection Tool
=============================================================
          ©2026 Athos García (athosworld.rf.gd)
The idea behind the algorithm was provided by Chemical-Shop8739
=============================================================
This program is designed to protect music (or other audio files)
from being used to train artificial intelligence (AI).

It uses a 5 stage audio spectral obfuscator that tackles the way
most modern AI music generators work, by converting audio into a
spectrogram.

Details about what exactly the stages mentioned do cannot be revealed 
as doing so may lead to a possible circumvention being developed.
=============================================================
                          USAGE
=============================================================
To use this tool, you must have an uncompressed WAV audio file at
a bit depth of (preferably) 32 bits. 

1. Click the "Open WAV" button on the upper panel to load the file.

2. Click the "Play" button to hear it played back.

3. Click the "Encode" button on the lower panel to start the encoding
process, you will be prompted to select a save location.

4. When the log text box displays "READY", you can now play back the
encoded file.
=============================================================
                          TIPS
=============================================================
Running the same audio file through the program will not result in
better protection and will only degrade the quality of it.
