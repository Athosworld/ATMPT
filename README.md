# ATMPT
Anti-Training Music Protection Tool. Uses 5 stages of spectral obfuscation to protect music from being used in AI training.

Most AI music models train on audio files by converting them into spectrograms, which is precisely what this tool attacks. It confuses training, making it hard to identify structures such as rythm or pitch while still sounding nearly the same to a human.

I can't make the source code for this tool public as it might lead to circumvention.
